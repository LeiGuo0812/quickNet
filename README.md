# quickNet

## Install

### Online install

```r
if (!require(devtools)) {
  install.packages('devtools')
}

devtools::install_github('LeiGuo0812/quickNet')
```

### Local install

Click Code --> download zip. Change your R working directory to where the zip is saved.

```r
if (!require(devtools)) {
  install.packages('devtools')
}

devtools::install_local('quickNet-main.zip')
```
